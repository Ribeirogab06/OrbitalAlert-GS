Perguntas Frequentes

Como os dados de trajetória são calculados?

As trajetórias são calculadas usando elementos orbitais TLE (Two-Line Element Set) fornecidos pelo NORAD e USSPACECOM. O modelo de propagação SGP4 é aplicado para prever a posição dos objetos com horizonte de até 7 dias.

Com que frequência os alertas são atualizados?

Os dados são atualizados a cada 4 horas. Para objetos com probabilidade de colisão acima de 0.001%, a frequência sobe para atualizações a cada hora.

O OrbitAlert possui API pública?

Sim, uma API RESTful está disponível para pesquisadores e instituições cadastradas. O acesso requer registro e aprovação da equipe técnica. Entre em contato via api@orbitalert.br.

Os dados cobrem apenas o Brasil?

O foco principal são objetos com passes sobre o território brasileiro e sua zona econômica exclusiva. Porém, o banco de dados global do USSPACECOM é utilizado como fonte, portanto objetos em órbita global também são catalogados.
